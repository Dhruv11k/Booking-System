# Booking Management System

A simple file-based Booking Management System built using Python.  
This project demonstrates basic backend logic, file handling, and menu-driven programming.

## Features
- Add new bookings
- View all bookings
- Search booking by ID
- Cancel booking
- Persistent storage using text files

## Tech Stack
- Python
- File Handling (Text Files)
